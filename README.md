Парсер объявлений авито с фильтрацией по городу и цене и сохранинием результата парсинга в Excel файл

Возможности парсера:
1.Поиск по любому запросу
2.Фильтрация по городу
3.Фильтр по ценам (в данной версии диапазон)
4.Парсинг нескольких страниц (зависит от лимита который установлен при запуске)
5.Очистка данных от мусора
6.Простые обходы защиты от ботов (задержки действий, настройки запросов)

Разбор все аргументов для запуска:
``` python
313(номер строчки)
collect_price # <- вызов самой функции парсера, можно изменить название при объявлении , но не менять на другие функции , иначе скрипт не будет работать( 
    q='iPhone 14 Pro Max',           # <- запрос поиска
    cond='б/у',                       # <-  состояние (пока не используется)
    city='Ростов-на-Дону',           # <-  город (или 'вся россия')
    price_range=(30000, 40000),      # <- диапазон цен (мин, макс)
    pages=5,                          # <- количество страниц
    output='iphone_results.xlsx',    # <- имя файла (None = автогенерация)
    price_tol=0.1                     # <- допустимое отклонение цены (10%)
)
```

Требования для запуска: 
1. Python 3.8+
2. Google Chrome
3. ChromeDriver (версия должна соответствовать установленному Chrome)
  Скачать: https://chromedriver.chromium.org/
  Добавить в PATH или указать путь в коде

Структура проекта:
```
    |-- parser.py <- файл с парсером, в конце которого происходит сам запуск  
    |-- requirements.txt <- файл с зависимостями
    |-- Readme.md <- документация 
```

Рук-во для страта:
1. Клонируйте репизоторий 
```bash
git clone <your-repo-url>
cd avito-parser
```
2. Установите все зависимости 
``` bash
pip install -r requirements.txt
```
3. Запуск (можете сразу поменять параметры запуска, на строке 313):
``` bash
python parser.py
```
